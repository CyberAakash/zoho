## Adding a New Doc

1. Create a new folder under `docs/` — e.g. `docs/my-new-topic/`
2. Add an `index.html` file inside it (copy the nav bar from an existing doc)
3. Add a new card entry in `docs/index.html` pointing to the new folder
4. Commit and push — GitHub Pages publishes automatically
